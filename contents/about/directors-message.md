# สารจากผู้อำนวยการ — Director's Message

สารจากผู้อำนวยการเปิดด้วยเรื่องเล่าสำหรับเด็ก: "The Very Hungry Caterpillar" เล่าถึงหนอนผีเสื้อตัวเล็กที่ฟักออกจากไข่ กินอย่างต่อเนื่องจนอ้วน แล้วหยุดเพื่อสร้างบ้านที่ปลอดภัยเรียกว่า "ดักแด้" เป็นเวลานาน

ผู้อำนวยการอธิบายว่าหนอนผีเสื้อรู้ว่าตัวเองไม่ได้ถูกสร้างมาเพียงเพื่อเป็นหนอนผีเสื้อหรือดักแด้ มันยอมรับการเปลี่ยนแปลงอย่างสมบูรณ์ (Metamorphosis) จากภายในสู่ภายนอก กลายเป็นผีเสื้อที่สวยงาม

คริสเตียนเปรียบเหมือนหนอนผีเสื้อตัวเล็ก พวกเขาได้รับอาหารฝ่ายวิญญาณจนเติบโตเป็นผู้ใหญ่ฝ่ายวิญญาณ แต่พระเจ้าไม่ได้ไถ่มนุษย์เพียง "เพื่อเป็นคริสเตียนที่ดี" รอการเสด็จกลับมาของพระคริสต์ พระเจ้าทรงปรารถนาการเปลี่ยนแปลงชีวิตอย่างสมบูรณ์ให้เป็นผู้รับใช้ของพระเจ้า

พระคริสตธรรมเชียงใหม่มุ่งมั่นที่จะมีส่วนในการเปลี่ยนแปลงชีวิตผ่านการฟื้นฟูอย่างสมบูรณ์ ผ่านการสอนที่ตั้งอยู่บนพระวจนะของพระเจ้า ทั้งภายในและภายนอกห้องเรียน เพื่อพัฒนา "คนงานที่ไม่ต้องอาย ใช้พระวจนะแห่งความจริงอย่างถูกต้อง" (อ้างอิง 2 ทิโมธี 2:15)
